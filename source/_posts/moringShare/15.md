* 人们常说金鱼的记忆只有短短的7秒，特别的短暂，所以吃了一粒饲料，7秒之后忘记了，又吃了一粒，总是容易撑到
* 这种叫做"健忘"。



|                  |                       |                   |
| ---------------- | --------------------- | ----------------- |
| 我的脑子一片空白 | My mind went blank.   | adj.空白的 n.空格 |
| 我健忘           | My memory is short.   | n.记忆            |
| 健忘             | short memory          |                   |
| 我记忆特别不错   | I have a good memory. |                   |
